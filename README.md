# 29th Street Deli & Grill

Website for 29th Street Deli & Grill, located at 29th & Dune Drive, Avalon, NJ.

## Overview

Single-file static website built with HTML, CSS, and vanilla JavaScript. No build tools or dependencies required — just open `index.html` in a browser or deploy directly to any static host.

## Deploying to Netlify

### Option A — Drag & Drop (easiest)
1. Go to [netlify.com](https://netlify.com) and log in
2. From your dashboard, drag the entire project folder onto the deploy drop zone
3. Done — Netlify assigns a URL instantly

### Option B — Connect to GitHub (recommended for ongoing updates)
1. Push this repo to GitHub
2. In Netlify: **Add new site → Import an existing project → GitHub**
3. Select this repo
4. Build settings:
   - **Build command:** *(leave blank)*
   - **Publish directory:** `.`
5. Click **Deploy site**

Future updates: push to GitHub → Netlify auto-deploys.

## Updating placeholder links

When your platforms are ready, find and replace these two placeholders in `index.html`:

| Placeholder | Replace with |
|---|---|
| `href="#" target="_blank" title="Online ordering coming soon"` | Your Toast POS ordering URL |
| `href="#" target="_blank" title="Catering platform coming soon"` | Your catering platform URL |
